Xcode 三方库管理工具 CocoaPods 的安装流程

2019-11-12

1. 移除现有 Ruby 默认源：

$ gem sources --remove https://rubygems.org/

2. 使用新的 Ruby 源：

$ gem sources -a https://gems.ruby-china.com

3. 查看是否替换成功：

$ gem sources -l

4. 安装 CocoaPods

$ sudo gem install cocoapods
